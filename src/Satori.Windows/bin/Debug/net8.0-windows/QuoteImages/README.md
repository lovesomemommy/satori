# Картинки цитат лотоса

Положите PNG с уже нанесённой цитатой в эту папку.

Имена файлов:
- `quote.lotus.01.png`
- `quote.lotus.02.png`
- …

Файлы копируются в output при сборке. Альтернативный путь: `assets/quotes/` рядом с exe.

Пока файла нет — показывается временная заглушка (только для разработки).
